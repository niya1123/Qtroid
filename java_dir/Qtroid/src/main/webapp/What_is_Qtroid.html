Qtroidとは？

エンジニアリングに関する知識を記録・共有するためのサービスであるQiita内にある記事の参考リンクが散見し見辛いので, 本アプリケーションでその参考リンクを収集し, 並べて表示することである.

how to

利用方法

  • 気になる技術名(プログラミング言語や人工知能など)が書かれたタグを選択
  • 選択したタグを持つ記事が「Qiita」内でのいいねが多い順にソートされる
  • ソートされた記事のURLが参考URLを共にリストで表示される

    • (※)タグが選択されていない場合では、ランキング一位の記事が表示されます
